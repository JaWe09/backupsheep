Databases

Connect any number of the databases for simple, secure, and cost-effective backups; locally or in the cloud.

Schedule frequent backups for MySQL, MariaDB and PostgreSQL. Backups are created using mysqldump & pgdump cli and you can also connect using SSH.

Web Servers

Connect any number of the cPanel, wordpress or other SFTP accounts for simple, secure, and cost-effective backups; locally or in the cloud.

Schedule backups for FTP,cPanel and WordPress servers using ftp/sftp protocol and store them in your own storage locations using storage integrations.
